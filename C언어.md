# 1-2
프로그램을 만드는 과정
1. 소스코드 작성
소스파일을 메모장(편집기)에서 만들고 하드디스크에 저장하면 text file로 저장됨(ASCII코드값으로 저장된 파일) = 소스코드
아스키코드 = 사람이 사용하는 기호를 컴퓨터 안에서 표현하는 방법 ex) 소스파일 A = 01000001 8bit(1byte)로
2. 컴파일
소스코드를 작성한다고 실행되지는 않음 명령어로 바꿔줘야함 = 컴파일
만들어진 파일 = 실행파일 
3. 실행
실행은 운영체제가 실행시킴
