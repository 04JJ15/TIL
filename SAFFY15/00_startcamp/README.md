# Markdown 기초 가이드

간단하고 가독성 좋은 문서 작성을 위한 경량 마크업 언어입니다. 아래 예시들은 필기(노트)를 마크다운으로 작성할 때 자주 쓰이는 문법들만 간단히 소개합니다.

---

## 1. 제목 (Headings)
입력 예시:
```
# 큰 제목 (h1)
## 중간 제목 (h2)
### 작은 제목 (h3)
```
결과:

# 큰 제목 (h1)
## 중간 제목 (h2)
### 작은 제목 (h3)

---

## 2. 강조 (Bold / Italic)
입력 예시:
```
**굵게** 또는 __굵게__
*기울임* 또는 _기울임_
```
결과: **굵게** / *기울임*

---

## 3. 목록 (Lists)
입력 예시:
```
- 순서 없는 항목 1
- 순서 없는 항목 2

1. 순서 있는 항목 1
2. 순서 있는 항목 2
```
결과:

- 순서 없는 항목 1
- 순서 없는 항목 2

1. 순서 있는 항목 1
2. 순서 있는 항목 2

---

## 4. 링크와 이미지
입력 예시:
```
[Google](https://www.google.com)
![대체텍스트](https://example.com/image.png)
```
결과: [Google](https://www.google.com)

이미지는 웹에 있는 이미지 주소(URL)를 넣어야 보입니다. 로컬 이미지를 깃허브에 올려서 참조할 수도 있습니다.

---

## 5. 코드 (inline / block)
입력 예시:
```
인라인 코드는 `printf()`처럼 백틱으로 감쌉니다.

```python
def hello():
    print("Hello, world!")
```
```
결과:

인라인 코드는 `printf()`처럼 백틱으로 감쌉니다.

```python
def hello():
    print("Hello, world!")
```

---

## 6. 인용문 (Blockquote)
입력 예시:
```
> 인용문 예시
```
결과:

> 인용문 예시

---

## 7. 표 (Table)
입력 예시:
```
| 이름 | 설명 |
| --- | --- |
| a | 첫 번째 항목 |
| b | 두 번째 항목 |
```
결과:

| 이름 | 설명 |
| --- | --- |
| a | 첫 번째 항목 |
| b | 두 번째 항목 |

---

## 8. 기타 유용한 것들
- 수평선: `---` 또는 `***`
- 체크박스(작업 목록): `- [ ]` 비어있음, `- [x]` 완료

예: 
- [x] 과제 읽기
- [ ] 예제 따라하기

---

## 팁
- 파일 확장자는 `.md` 사용하세요 (예: `notes.md`).
- 깃허브나 VS Code에서 바로 미리보기가 가능합니다.
- 온라인 에디터: StackEdit, Dillinger 등을 사용하면 편리합니다.

필기용으로는 위 문법들만 알면 충분합니다. 더 배우고 싶으면 공식 문서나 깃허브 마크다운 가이드를 참고하세요.
