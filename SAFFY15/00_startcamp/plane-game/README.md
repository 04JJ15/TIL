# Plane Flyer — 간단 웹게임

비행기를 조종해 장애물을 피하는 간단한 웹게임입니다. 마우스(또는 터치)를 누르고 있으면 비행기가 상승하고, 놓으면 중력으로 하강합니다.

실행 방법

1. 파일 탐색기에서 [plane-game/index.html](plane-game/index.html)를 더블클릭해 브라우저로 엽니다.
2. 화면을 클릭(또는 터치)하면 게임이 시작됩니다. 누르고 있는 동안 상승, 놓으면 하강합니다.

구조

- `index.html` — 게임 캔버스와 레이아웃
- `style.css` — 간단한 스타일
- `script.js` — 게임 로직 (물리, 충돌, 스폰, 렌더링)

확장 아이디어

- 점점 빨라지는 난이도, 파워업 추가
- 이미지/사운드 리소스 추가
- 모바일 터치 최적화
